# SSHMonitor
Monitor de usuarios en Linea en SSH


## Instalación:

```
wget -O /bin/Monitor https://raw.githubusercontent.com/powermx/SSHMonitor/master/Monitor.sh
chmod +x /bin/Monitor
```

## Uso:
```
Monitor
```

Este script monitorea los usuarios tanto Dropbear como OpenSSH en nuestro VPS.
